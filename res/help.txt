
USAGE:
    python3 cli.py --help | -h | -?
    python3 cli.py --url_file ~/my_file.txt
    python3 cli.py --uf ~/my_file.txt

    --help, -h -?
        Display this help message.
    --url_file, --uf
        Specify filepath of URL list file to use.

DETAILS: 
    Target .txt files should contain one RSS feed URL per line with no blank lines.

    www.myurl.com/rss
    www.myurl2.com/rss
    www.myurl3.com/rss
